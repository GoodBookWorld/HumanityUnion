# EPIC_02_MEMBER_PROFILE

## Capability 01 — Human Identity

Humanity Union Platform

Epic Specification

Version 1.0

---

# Purpose

Epic 02 establishes the Member Profile subsystem of the Humanity Union Platform.

Its purpose is to give every Member a clear, editable, and human-centered profile foundation.

The profile is not only a user account page.

It is the first visible expression of a Member's civic identity inside Humanity Union.

---

# Mission

Enable Members to view, manage, and gradually shape their Humanity Union identity while preserving the separation between technical identity and civic participation.

Authentication identifies access.

Member Profile represents the person as a participant.

---

# Scope

Epic 02 includes:

* Member Profile API
* Profile read model
* Profile update model
* Profile web page
* Public profile foundation
* Profile visibility preparation

This Epic does not include:

* real database persistence
* password management
* full privacy system
* verification workflow
* Fair Engine
* My Impact dashboard
* public directory search
