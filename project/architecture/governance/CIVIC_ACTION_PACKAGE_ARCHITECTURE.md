# Purpose

Define the **Civic Action Package (CAP)** — the structured public civic document issued automatically when a Collective Decision closes.

CAP is not a letter, email, or petition. It is the official public record carrying Humanity Union’s constitutional decision result and the civic history behind it.

---

# Core Rules

1. One Collective Decision result → one Civic Action Package.
2. CAP is generated automatically when a Collective Decision closes.
3. CAP is public, reference-based, and immutable after issue.
4. CAP delivery to recipients is deferred to TASK-040.

- Official response intake
